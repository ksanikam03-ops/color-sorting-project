Project Overview

This project uses Raspberry Pi 4, a USB camera, OpenCV, a proxy sensor, conveyor belt and Dobot Magician to detect Red, Blue and Yellow objects and automatically sort them into the correct bins.

Features

Computer Vision
OpenCV Color Detection
Automatic Conveyor
Proxy Sensor
Dobot Pick & Place
Live Dashboard

Robot Calibration

  1. Run calibration mode.
  2. Teach PICK position.
  3. Teach BIN_RED.
  4. Teach BIN_BLUE.
  5. Teach BIN_YELLOW.
  6. Save coordinates.
